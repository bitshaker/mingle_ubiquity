h2. Ubiquity Commands for Mingle

A set of "Ubiquity":https://wiki.mozilla.org/Labs/Ubiquity/ commands for "Mingle":http://studios.thoughtworks.com/mingle-agile-project-management/ users.

I'm open to any feature requests and patches.

h3. Current Commands

* mingle-grid-find: Shows Mingle cards in the grid view based on your search

h3. Planned Commands

* mingle-new: Create a new card based on the type in the command
* mingle-search: Search the current project
* mingle-list-find: Show only lines that have specified search text (same as mingle-grid-find)
